# protoscmn

Protobuf definations used for cross languages.

in protobuf3

## Compile

```shell
build -h # help
build -g # to golang
build -p # to python
build -c # clean
```

## Desciption

- errors.proto
> All types of error codes

- confs.proto
> All kinds of configures

- msgq_*.proto
> message queue data format for backend usage.
